App de prueba de concepto orientada a red team y aprendizaje de seguridad en Android. Implementa y permite experimentar con distintos vectores de interacción entre Activities:

- Actividades exportadas
- Generación y envío de Intents
- Hijack de Intents implicitos
- Mecanismos de respuestas entre actividades
- Pending Intents Mutables
- Deeplinks
